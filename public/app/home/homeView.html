Projects

  • Pictured below are my 3D printers. Both of these printers are hand built reprap 3d printers based on the prusai3 layout. The first printer is a laser cut acrylic frame standard prusai3 with a few custom parts and was my first foray into 3D printing. The second printer is much more sophisticated which I built to help me with my masters research. Both printers are web enabled and are tuned well enough to produce models which rival much more expensive machines.

    • Technical Specifications
    • 200x200x170mm print area
    • 37mm/sec max printing speed
    • .15mm Z resolution
    • Active bed leveling with inductive sensor
    • Arduino based motor controller and Raspberry pi web interface
    • Technical Specifications
    • 200x200x280mm print area
    • ~110mm/sec max printing speed
    • ~.08mm Z resolution (sub 100 micron!)
    • Active bed leveling with inductive sensor
    • Arduino based motor controller and Raspberry pi web interface
    • PID controlled air envelope with enclosure (up to 60°C)
    • Dual power supply (dedicated heater power supply)
